常見問題 創作理念

Q1 table for ONE 為什麼出現？

在汲汲營營、飛簷走壁、各種責任身份多重扮演、照鏡子時真是一股陌生感阿 的生活中，table for ONE 是個能讓你、我、他，在喧囂的城市中，暫緩腳步、關機片刻、跳脫日常的一人體驗空間。暫時拋下一切，用一頓飯的時間，享受屬於一個人的好時光吧！

一個人很好，享受它。

Q2 請問在 table for ONE ，我可以期待什麼樣的「一人飲食體驗」呢？

在 table for ONE，你將擁有一張大小適中的個人餐桌、一套完整的餐點，與23位陌生人的陪伴（當然，有坐滿的話）。除此之外，在 table for ONE 的空間中，也有許多樂趣等著你 - 只要你留心觀察、勇敢發掘，這裡將不再只是一場體驗、一場飯局、一場活動 - 它將會是在最平凡的生活中，最預料之外的一次一人之旅。

Q3 請問餐點內容是什麼呢？

餐點內容為來店體驗的一環，故官網上與社群上僅提供概念菜單。但！為維持體驗的新鮮感（比如網路上已露出的餐點照片降低了神秘感），table for ONE 將會不定時調整食材內容。若有特殊飲食限制者（茹素、對甲殼類過敏等），建議採用線上預付訂位並事先註明飲食習慣，我們將會盡力為您調整餐點。

Q4 整場體驗時間大概多久呢？

體驗時間取決於你的步調，大約介於聽完一張國語專輯的60分鐘至看完一部經典港產片的90分鐘。也擔心有客人像是觀賞哈利波特七部曲般的從天黑到打烊，所以我們表定90分鐘為限。

Q5 請問可以與朋友一同前往嗎？

您能與朋友一同前來。table for ONE 僅提供單人訂位，但您可以與朋友預訂同一時段，屆時一同前來。table for ONE 為一人飲食體驗，故不提供併桌或是兩桌靠近一點的服務。為了體驗品質，您與您的朋友也將會被安排於視線不易相互交會的座位。

Q6 請問table for ONE僅限單身的人報名嗎？

已婚、未婚、單身、穩定交往中、一言難盡、開放式戀愛中 - 只要你想要與自己吃頓飯，table for ONE 開放給任何需要或想要一個人的你。

Q7 請問我可以在這裡認識新朋友嗎？

在不影響他人體驗品質的前提下，若客人們有意眉來眼去、飛鴿傳書、隔空一起乾五杯shots等各種互動，table for ONE不設限。現場服務人員有權力柔性勸導、適度制止過於影響他人之來賓。

Q8 請問有特別的服裝需求嗎？

沒有用餐服裝的需求，西裝、套裝、泳裝，以自己最舒服的姿態出現都好。
當然，過於裸露或是根本就是完全裸露，無論男性女性我們都會稍稍注意一下。

Q9 在體驗過程中，有什麼特別的限制嗎？可以用手機看劇、看書、或是工作嗎？

體驗過程中會請來賓把手機與電子產品調整成飛航模式，並希望來賓避免積極對外聯繫，以及頻繁使用手機或其他電子產品等。我們相信在獨自一人時，是更能體會、觀察、探索眼前的各種美好，就像是在國外旅遊時沒了3G網路，反而更能留心在街頭上所經歷的種種當下呢。

table for ONE不會特別設限來賓的任何行為(當然以不影響他人為前提)，相比於一如平時生活般的吃飯配劇、配本書、配著晚點要交的報告，我們更希望來賓們能體驗一個人的當下，與自己的一餐。

Q12 請問若想要更改日期，我該怎麼做？

於用餐日三天前，來電或來信告知需求，皆能更改訂位日期。惟用餐日前兩天至當日皆不接受日期更改，建議來賓可將座位轉讓給朋友或家人。有任何特殊突發狀況，歡迎來信、來電與我們聯繫。

Q13 請問取消訂位與退費機制機制為何？

於用餐日三天前取消訂位，於扣除金流系統手續費 5% 後，將獲全額退費；於用餐日兩天前取消訂位，因廚房已完成備料，扣除金流系統手續費 5% 與廚房備料成本，將退回 50% 之費用。用餐當日未出席、也未事先來信或來電告知者，table for ONE 將不進行退費。有任何特殊突發狀況，歡迎來電與我們聯繫。

對於「一個人的好」之觀察

工作、應酬、聚會、派對，日復一日，我們總被有意無意地鼓勵社交。個性外向、善於交際、有趣健談等形容詞，總是比內向、安靜、害羞更來的讓人嚮往。於是，在強調人際關係的社會中，「獨自一人」的常被掛上寂寞、無趣、邊緣人等標籤，但屬於「一個人的好」，卻總鮮少被提及與讚揚。

有趣的是，在心理學與社會學的研究中，獨自一人 (Alone) 與獨處 (Solitude) 對沈澱、思考、想像力、創造力、整合能力都有相當正面的影響；而一個人也與自由、深刻、啟迪等積極字詞有所關聯。換句話說，獨自一人是重要的，並與社交一樣重要。在繁忙的生活中，短暫跳脫各種責任、身份，擁有「屬於自己一人的時間」更是相當重要的喘息與充電。

有鑑於此，驚喜製造決定為節奏快速、行程緊湊的都市人，提供一個特別為「一個人」打造的硬體空間，並以有探索及驚喜為核心，設計出一個人也能把玩的軟性體驗。來這裡放鬆、沈澱、短暫跳脫一下生活吧！一個人很好，而且值得一個專門的空間告訴你有多好。

對於你、我、他「在生活中，一個人用餐」的觀察

路邊小店簡單快速，一個人解決一餐最適合了。時間若寬裕些，找間拉麵店或簡單連鎖店鑽進去，與人們相互挨著，在小小的座位上吃著漂亮點的料理。但更多時候，總本能地與老闆說聲外帶，回家佐著手機電腦吃。

這些都是一個人吃飯的日常，簡單、快速，為了飽食的日常。

在節奏越趨快速的時代，一人用餐的需求越來越高，陸續有餐廳開始設立一人座位，但整體還是以簡單、快速為概念，個人用餐空間小，餐點選擇也以單點飽食的便當、丼飯、拉麵為主。

而以「讓一個人能好好坐下來，慢慢舒服享用一個完整套餐」、「用一頓飯的時間，好好與自己相處」為初衷的一人餐飲形式，在日常用餐選擇中也屬少見。因此，驚喜製造選擇以「一人用餐」、「舒適的硬體」、「有趣的軟體」為核心，推出 table for ONE 一人餐桌，給在生活中時不時會一人吃飯的我們，另一種能夠慢下來的選擇、另一種與自己好好相處的選擇。
